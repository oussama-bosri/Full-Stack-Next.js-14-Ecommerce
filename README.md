## Technologies Used

- **Next.js 14**: The React framework for production.
- **React.js**: A JavaScript library for building user interfaces.
- **Strapi CMS**: Headless CMS to manage content.
- **Tailwind CSS**: A utility-first CSS framework for rapid UI development.
- **Stripe Payment**: Integration for handling online payments.
- **Github Actions**: Continuous Integration and Continuous Deployment (CI/CD).

## Features

- User authentication and authorization.
- Product listings and detailed product pages.
- Shopping cart and checkout process.
- Order management.
- Payment processing with Stripe.
- Responsive design with Tailwind CSS.
- Content management with Strapi CMS.
- Continuous Integration with Github Actions.
